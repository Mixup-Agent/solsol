# Solar Pro3 AI Agent 해커톤 대회 안내

## 개요

본 대회는 **Solar Pro3**를 활용하여 실제 문제 해결이 가능한 AI Agent를 개발하는 것을 목표로 합니다.

- 문제 정의 → 에이전트 기획 → MVP 구현 → 시연·발표의 전 과정을 경험
- 단순 기능 구현을 넘어 **기획 → 구현 → 검증 → 발표**까지 이어지는 실무형 AI 서비스 개발

## 주제

**Solar Pro3를 활용한 Agent 개발**

### 세부 주제 예시 (자유롭게 확장 가능)

1. 이메일, 문서, 일정 등을 통합 관리하는 **업무 자동화 Agent 시스템**
2. 사용자 요청만으로 분석부터 실행까지 수행하는 **End-to-End AI Agent**
3. 다양한 API를 연결해 작업을 수행하는 **스마트 워크플로우 Agent**
4. 여러 Agent가 협력하여 문제를 해결하는 **Multi-Agent 협업 시스템**
5. 특정 산업 문제를 해결하는 **도메인 특화 AI Agent 서비스**
6. 반복 업무를 자동화하는 **개인 생산성 향상 Agent**

## 개발 규칙

- **Solar Pro3 API**를 활용한 Agent 기능이 핵심적으로 포함되어야 함
- 단순 챗봇이 아닌, 사용자 요청을 기반으로 **판단·작업 수행하는 Agent 구조** 권장
- 외부 API, 오픈소스 라이브러리, 프레임워크 사용 가능
  - 단, 사용한 기술은 **발표 자료 또는 GitHub README에 명시** 필수
- 최종 결과물은 **현장 시연 가능한 MVP** 형태여야 함
- 구현 결과는 **제출한 GitHub 링크를 통해 확인** 가능해야 함
